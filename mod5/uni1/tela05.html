O que diferencia o planejamento de uma "intervenção na escola" de outros planejamentos que realizamos no contexto escolar, como os planos de aula e planos de ensino?

  • ❯❯O motivo!

Projeto de intervenção é uma estratégia para intervir sobre e tentar resolver uma situação-problema.

O projeto de intervenção é um plano de ação para o enfrentamento de um obstáculo específico, num dado contexto. Mesmo que a intervenção considere e contribua para o enfrentamento de problemas estruturais e complexos, o projeto deve ter objetivos modestos. A intervenção é apenas um passo – mesmo que de formiguinha! – que deve trazer soluções efetivas para o problema em foco (como aprofundaremos na Unidade 3).
